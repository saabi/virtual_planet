# fe.old — legacy Sapper planet editor

Archived **Sapper / Svelte 3** app moved here during the SvelteKit 2 migration (2026-06-16).

- Default panel: `PlanetDisplay` (`src/routes/index.svelte`)
- Planet GLSL: `src/shaders/planet.frag`, `planet.vert`
- Presets: `src/planet-editor/sample-planets.js`

```sh
npm install
npm run dev
```

Do not extend this tree for new features — port into `fe/` instead.
