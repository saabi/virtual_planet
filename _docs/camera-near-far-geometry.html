Near / Far Camera Geometry

How a perspective camera defines a truncated viewing pyramid (frustum), how WebGPU maps depth into [0, 1], and how Virtual Planet chooses near/far distances for focused-body views vs multi-body scene views.

1. Side view — frustum between near and far

The camera sits at the eye. Rays through the viewport edges form a truncated pyramid. Geometry closer than near or farther than far is clipped. Only the shaded volume is rasterized with correct depth.

2. Top view — frustum cross-section

Horizontal field of view scales with aspect ratio. Near/far rectangles grow with distance from the eye along the view axis.

3. WebGPU depth mapping

After projection, clip-space z is in [0, 1] (not OpenGL’s [-1, 1]). A screen pixel ray unprojects NDC z=0 at the near plane and z=1 at the far plane. 

// WebGPU perspective (z ∈ [0, 1]) — orbitCamera.perspective()
f = 1 / tan(fovY / 2)
P[2,2] = far / (near - far)
P[2,3] = -1
P[3,2] = near·far / (near - far)

// Picking ray — runtime-cpu pointerRay()
origin = unproject(ndcX, ndcY, 0, invViewProj)  // on near plane
farPt  = unproject(ndcX, ndcY, 1, invViewProj)  // on far plane
direction = normalize(farPt - origin)

4. Virtual Planet near/far policies

Two complementary strategies: distance-scaled defaults for a single focused body, and scene-bounds fitting when many bodies share one depth buffer. 

// Single focused body — fe/lib/planet/camera/orbitCamera.ts
focusedBodyNearFar(d) => [ max(1, d * 0.002), d * 20 ]

// Multi-body scene — fe/lib/planet/scene3d/orbitCamera.ts
sceneNearFar(eye, bodies, extraPoints, farExtents)
  nearD = min over bodies of (|eye − center| − radius)
  farD  = max over bodies of (|eye − center| + radius)
  near  = max(NEAR_FLOOR, nearD * 0.5)
  far   = max(farD * 1.05, near * 4)
  if (far / near > MAX_DEPTH_RATIO) far = near * MAX_DEPTH_RATIO

5. Frustum planes (six half-spaces)

The view-projection matrix defines six inward-facing planes: left, right, bottom, top, near, far. Culling tests whether a sphere/box lies entirely outside any plane.
